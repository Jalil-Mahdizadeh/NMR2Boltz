# Primary standards and implementation references

The implementation and methods document were checked against these primary or project-maintained sources.

1. NMR Exchange Format (NEF), version 1.1 specification and commented examples, audited revision `9ab6bc023a406c87df407837597efecaf289fe55`: https://github.com/NMRExchangeFormat/NEF
2. BMRB NMR-STAR dictionary, audited revision `35c6e32a4c948de8d9bf8b367dfab1217a216c0a`: https://github.com/bmrb-io/nmr-star-dictionary
3. BMRB-maintained PyNMRSTAR parser: https://github.com/bmrb-io/PyNMRSTAR
4. BMRB REST API documentation and entry service: https://api.bmrb.io/v2/
5. wwPDB NMR/NEF translation utilities and NEF-to-NMR-STAR equivalence tables: https://github.com/wwPDB/py-wwpdb_utils_nmr
6. wwPDB Chemical Component Dictionary: https://www.wwpdb.org/data/ccd
7. Gemmi CIF/STAR parser documentation: https://gemmi.readthedocs.io/en/latest/cif.html
8. Upstream Boltz implementation: https://github.com/jwohlwend/boltz
9. BoltzUI atom-contact implementation used as the target schema, audited revision `c3e5c7f6ae80d9261c357a0951a0929c50a1115d`: https://github.com/Jalil-Mahdizadeh/BoltzUI
